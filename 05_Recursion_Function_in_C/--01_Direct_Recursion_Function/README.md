# 📘 Direct Recursion in C Programming

## 🚀 Introduction

Direct recursion is a type of recursion in which a function calls itself directly to solve a problem.

It is the most basic and commonly used form of recursion in C programming.

---

## 🎯 Why Direct Recursion is Important?

Direct recursion is widely used in:

- Mathematical problems ➗  
- Tree traversal 🌳  
- Sorting algorithms 🔍  
- Competitive programming 💻  
- Interview coding questions 💼  

---

## 🌍 Real Life Example

Think of looking into two parallel mirrors 🪞

The reflection keeps repeating itself in the same direction.

This is similar to direct recursion → same function calling itself repeatedly.
