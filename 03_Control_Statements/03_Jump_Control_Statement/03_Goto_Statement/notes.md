# Goto Statement - Quick Notes

---

## Definition

Transfers control to labeled statement.

---

## Syntax

```c
goto label;

label:
```

---

## Example

```c
goto end;

end:
printf("Done");
```

---

## Key Points

✅ Unconditional jump  
✅ Uses label  
❌ Avoid excessive use  
✅ Used in error handling  

---

## Output Concept

Program jumps directly to label section.
