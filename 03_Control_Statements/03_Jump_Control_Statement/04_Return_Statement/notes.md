# Return Statement - Quick Notes

---

## Definition

Used to return value from function.

---

## Syntax

```c
return value;
```

---

## Example

```c
int add()
{
    return 5 + 3;
}
```

---

## Key Points

✔ Ends function  
✔ Sends value back  
✔ Must match return type  

---

## Types

✔ return value  
✔ return (void functions)
