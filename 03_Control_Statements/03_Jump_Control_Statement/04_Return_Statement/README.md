# 📖 Introduction

The **return statement** in C is used to:

```txt
Send value back from a function to the calling function
```

It also:

```txt
Ends the execution of a function immediately
```

---

# 🧠 Why Use Return Statement?

We use return to:

- Send result back to main()
- Stop function execution
- Store computed values
- Build reusable logic

---

# 🏗 Syntax

```c
return value;
```

or

```c
return;
```

(for void functions)

---

# ⚙ Working

When return is executed:

```txt
Function stops immediately
Value is sent back to caller
```
