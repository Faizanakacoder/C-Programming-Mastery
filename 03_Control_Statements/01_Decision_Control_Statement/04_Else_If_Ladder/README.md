# Else If Ladder in C

## Overview

The `else if ladder` is used when multiple conditions need to be checked.

It executes only one block of code from multiple conditions.

Syntax:

```c
if(condition1)
{
    // code
}
else if(condition2)
{
    // code
}
else
{
    // code
}
```
