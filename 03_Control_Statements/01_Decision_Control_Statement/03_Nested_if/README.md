# Nested If Statement in C

## Overview

A `Nested If` statement means an `if` statement inside another `if` statement.

It is used when multiple conditions need to be checked one after another.

Syntax:

```c
if(condition1)
{
    if(condition2)
    {
        // code
    }
}
```
