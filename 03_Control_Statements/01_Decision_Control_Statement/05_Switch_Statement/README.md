# Switch Statement in C

## Overview

The `switch` statement is used for multiple condition checking.

It allows a program to choose one block of code from many options.

Syntax:

```c
switch(expression)
{
    case value1:
        // code
        break;

    case value2:
        // code
        break;

    default:
        // code
}
```
