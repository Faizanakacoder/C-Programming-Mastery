# If Statement in C - Quick Notes

## Definition

`if` statement is used for decision making.

It executes code only when condition is true.

---

## Syntax

```c
if(condition)
{
    statement;
}
```

---

## Example

```c
if(age >= 18)
{
    printf("Eligible");
}
```

---

## Important Point

If condition is:

```txt
True  → Executes
False → Skips
```

---

## Short Summary

The `if` statement executes code only when condition is true.
