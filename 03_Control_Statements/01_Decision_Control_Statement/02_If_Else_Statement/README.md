# If Else Statement in C

## Overview

The `if-else` statement is used for decision making in C programming.

It executes one block of code if the condition is **true** and another block if the condition is **false**.

Syntax:

```c
if(condition)
{
    // code if true
}
else
{
    // code if false
}
```
