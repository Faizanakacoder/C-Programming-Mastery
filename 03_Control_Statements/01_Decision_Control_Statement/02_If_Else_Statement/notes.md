# If Else Statement in C - Quick Notes

## Definition

`if-else` statement is used for decision making.

It executes one block if condition is true and another if false.

---

## Syntax

```c
if(condition)
{
    statement;
}
else
{
    statement;
}
```

---

## Example

```c
if(age >= 18)
{
    printf("Eligible");
}
else
{
    printf("Not Eligible");
}
```

---

## Important Point

```txt
True  → if block executes
False → else block executes
```

---

## Short Summary

`if-else` statement is used when there are two possible outcomes.
