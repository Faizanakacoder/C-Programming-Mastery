# While Loop in C - Quick Notes

## Definition

The `while` loop repeats code while condition is true.

---

## Syntax

```c
while(condition)
{
    statement;
}
```

---

## Example

```c
int i = 1;

while(i <= 5)
{
    printf("%d", i);
    i++;
}
```

---

## Important Point

```txt
Condition checks first
```

This is called:

```txt
Entry Controlled Loop
```

---

## Short Summary

While loop repeats code until condition becomes false.
