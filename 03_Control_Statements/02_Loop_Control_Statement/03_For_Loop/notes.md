# For Loop in C - Quick Revision Notes

---

## Definition

For loop repeats code for a fixed number of times.

---

## Syntax

```c
for(initialization; condition; update)
{
    statements;
}
```

---

## Example

```c
for(i = 1; i <= 5; i++)
{
    printf("%d\n", i);
}
```

---

## Type of Loop

```txt
Entry Controlled Loop
```

---

## Important Parts

```txt
Initialization
Condition
Update
```

---

## Important Points

✅ Condition checks first  
✅ Best for fixed repetition  
✅ Compact syntax  
✅ Commonly used loop

---

## Short Summary

For loop repeats code for a known number of times.
