# Comments in C - Quick Notes

## Definition

Comments are explanatory notes written inside a program.

The compiler ignores comments during execution.

---

## Types of Comments

### 1. Single Line Comment

Syntax:

```c
// Comment
```

Example:

```c
// Printing message
printf("Hello");
```

---

### 2. Multi Line Comment

Syntax:

```c
/*
Comment Line 1
Comment Line 2
*/
```

Example:

```c
/*
This program
prints message
*/
printf("Hello");
```

---

## Advantages of Comments

- Improves readability
- Explains program logic
- Helps debugging
- Makes maintenance easier

---

## Important Point

Comments do not affect program execution because the compiler ignores them.

---

## Short Summary

Comments are used to explain code and improve readability in C programming.
