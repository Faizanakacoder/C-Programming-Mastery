# Constants in C - Quick Notes

## Definition

Constants are fixed values that cannot be changed during program execution.

---

## Types of Constants

### 1. Integer Constant

Example:

```c
10
100
```

---

### 2. Floating Constant

Example:

```c
10.5
20.75
```

---

### 3. Character Constant

Example:

```c
'A'
'B'
```

---

### 4. String Constant

Example:

```c
"Hello"
```

---

## const Keyword

Syntax:

```c
const data_type variable = value;
```

Example:

```c
const float PI = 3.14;
```

---

## Advantages

- Stores fixed values
- Improves readability
- Prevents accidental changes

---

## Short Summary

Constants are fixed values that remain unchanged during program execution.
