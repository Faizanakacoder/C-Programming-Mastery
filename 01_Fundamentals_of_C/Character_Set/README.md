# Character Set in C

## Overview

A Character Set in C refers to the collection of valid characters that can be used to write programs.

These characters are used to create:

- Keywords
- Variables
- Constants
- Operators
- Symbols

Without a character set, writing a C program is impossible.

---

## Types of Characters in C

1. Alphabets
2. Digits
3. Special Symbols
4. White Spaces
