# Character Set in C - Quick Notes

## Definition

Character Set is the collection of valid characters used in a C program.

---

## Types of Character Set

### 1. Alphabets
- A to Z
- a to z

Example:

```c
int age;
```

---

### 2. Digits
- 0 to 9

Example:

```c
int num = 100;
```

---

### 3. Special Symbols

Examples:

```txt
+ - * / % = ; { } ( ) #
```

---

### 4. White Spaces

Types:
- Space
- Tab (`\t`)
- New Line (`\n`)

---

## Importance

- Helps create valid programs
- Makes code readable
- Used for variables and operations

---

## Short Summary

Character Set is the collection of alphabets, digits, symbols, and spaces used to write C programs.
