# Theory - Character Set in C

## Introduction

A Character Set is a collection of valid characters that are allowed in a C program.

These characters help programmers write instructions in C language.

Example:

```c
int age = 20;
```

This statement contains alphabets, digits, symbols, and spaces.

---

## Types of Character Set

### 1. Alphabets

C language uses:

- Uppercase letters → A to Z
- Lowercase letters → a to z

Example:

```c
int age;
```

---

### 2. Digits

Digits from `0 to 9` are used.

Example:

```c
int num = 100;
```

---

### 3. Special Symbols

Special symbols are used for operations and syntax.

Examples:

```txt
+  -  *  /  %  =  ;  { }  ( )  #
```

Example:

```c
a + b;
```

---

### 4. White Spaces

White spaces improve readability.

Examples:

- Space
- Tab
- New Line

Example:

```c
int age = 20;
```

---

## Importance of Character Set

- Helps write valid programs
- Improves readability
- Supports variables and operators
- Makes program structure possible

---

## Conclusion

Character Set is the foundation of C programming because every C program is built using valid characters.
