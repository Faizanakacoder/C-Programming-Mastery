# Keywords in C - Quick Notes

## Definition

Keywords are reserved words with predefined meanings in C.

Keywords cannot be used as variable names.

---

## Examples of Keywords

```txt
int
float
char
if
else
return
for
while
```

---

## Common Keywords and Uses

| Keyword | Use |
|----------|-----|
| int | Integer type |
| float | Decimal type |
| char | Character type |
| if | Condition |
| return | Return value |

---

## Important Rules

- Keywords are reserved words
- Cannot be used as variable names
- Have fixed meaning
- Usually written in lowercase

---

## Example

```c
int age = 20;
```

Here, `int` is a keyword.

---

## Short Summary

Keywords are reserved words used to write programs in C.
