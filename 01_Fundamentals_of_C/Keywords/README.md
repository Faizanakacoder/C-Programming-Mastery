# Keywords in C

## Overview

Keywords are reserved words in C programming.

They have predefined meanings and cannot be used as variable names.

Example:

```c
int age = 20;
```

Here, `int` is a keyword.

---

## Examples of Keywords

Some common C keywords are:

- `int`
- `float`
- `char`
- `if`
- `else`
- `return`
- `for`
- `while`
