# Theory - Keywords in C

## Introduction

Keywords are reserved words in C programming language.

They have predefined meanings and special purposes.

Keywords cannot be used as variable names.

Example:

```c
int age = 20;
```

Here, `int` is a keyword.

---

## Features of Keywords

- Predefined meaning
- Cannot be changed
- Cannot be used as identifiers
- Used to write program logic

---

## Common Keywords in C

| Keyword | Purpose |
|----------|----------|
| int | Integer data type |
| float | Decimal values |
| char | Character values |
| if | Conditional statement |
| else | Alternative condition |
| return | Returns value from function |
| for | Looping |
| while | Repetition |

---

## Why Keywords are Important?

Keywords are essential because they define:

- Data types
- Conditions
- Loops
- Function behavior

Without keywords, writing programs in C is impossible.

---

## Rules of Keywords

- Keywords cannot be used as variable names.
- Keywords are lowercase.
- Keywords have fixed meaning.

Incorrect Example:

```c
int int = 10;
```

Correct Example:

```c
int age = 10;
```

---

## Conclusion

Keywords are reserved words used to write valid C programs.
