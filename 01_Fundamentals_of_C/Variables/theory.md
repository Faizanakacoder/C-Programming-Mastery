# Variables in C

## Definition

A variable is a named memory location used to store data.

---

## Syntax

```c
data_type variable_name;
```

Example:

```c
int age;
float salary;
char grade;
```

---

## Rules for Naming Variables

- Must begin with letter or underscore
- Cannot contain spaces
- Cannot use keywords
- Variable names are case sensitive

---

## Types of Variables

- Integer
- Float
- Character
- Double

---

## Real Life Example

Variables work like containers that store information.
