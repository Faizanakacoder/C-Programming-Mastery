# Tokens in C

## Overview

Tokens are the smallest individual units in a C program.

A C program is made up of different types of tokens.

Example:

```c
int age = 20;
```

In this statement:

- `int` → Keyword
- `age` → Identifier
- `=` → Operator
- `20` → Constant
- `;` → Special Symbol

These are called tokens.

---

## Types of Tokens in C

1. Keywords
2. Identifiers
3. Constants
4. Operators
5. Special Symbols
