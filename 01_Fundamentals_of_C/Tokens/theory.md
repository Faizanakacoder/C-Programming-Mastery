# Theory - Tokens in C

## Introduction

Tokens are the smallest units of a C program.

Every C statement is made up of tokens.

Example:

```c
int age = 20;
```

This statement contains different tokens.

---

## Types of Tokens

### 1. Keywords

Keywords are reserved words with predefined meanings.

Example:

```c
int
float
char
```

---

### 2. Identifiers

Identifiers are names given to variables, functions, or arrays.

Example:

```c
age
marks
salary
```

---

### 3. Constants

Constants are fixed values.

Example:

```c
10
20.5
'A'
```

---

### 4. Operators

Operators are symbols used to perform operations.

Example:

```c
+
-
*
/
=
```

---

### 5. Special Symbols

Special symbols are used for syntax.

Example:

```txt
; { } ( ) #
```

---

## Importance of Tokens

- Helps understand program structure
- Makes code meaningful
- Required to write valid programs

---

## Example of Tokens

```c
int number = 10;
```

| Token | Type |
|--------|------|
| int | Keyword |
| number | Identifier |
| = | Operator |
| 10 | Constant |
| ; | Special Symbol |

---

## Conclusion

Tokens are the building blocks of a C program.
