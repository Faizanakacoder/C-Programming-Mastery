# Tokens in C - Quick Notes

## Definition

Tokens are the smallest units of a C program.

---

## Types of Tokens

### 1. Keywords

Example:

```c
int
float
char
```

---

### 2. Identifiers

Example:

```c
age
marks
salary
```

---

### 3. Constants

Example:

```c
10
20.5
'A'
```

---

### 4. Operators

Example:

```txt
+ - * / =
```

---

### 5. Special Symbols

Example:

```txt
; { } ( ) #
```

---

## Example

```c
int age = 20;
```

| Token | Type |
|--------|------|
| int | Keyword |
| age | Identifier |
| = | Operator |
| 20 | Constant |
| ; | Special Symbol |

---

## Short Summary

Tokens are the smallest building blocks of a C program.
