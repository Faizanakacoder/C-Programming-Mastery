# 📘 Indirect Recursion in C Programming

## 🚀 Introduction

Indirect recursion is a type of recursion in which a function does NOT call itself directly.

Instead, one function calls another function, and that function calls the first function again.

This creates a **cycle of function calls**.

---

## 🎯 Why Indirect Recursion is Important?

Indirect recursion is used in:

- Complex state-based problems 🔄  
- Parsing systems 📄  
- Compiler design 🧠  
- Menu-driven programs 🖥️  
- Advanced algorithm design 💡  

---

## 🌍 Real Life Example

Think of a relay race 🏃‍♂️:

- Runner A passes baton to Runner B  
- Runner B passes it back to Runner A  

This back-and-forth process is indirect recursion.
