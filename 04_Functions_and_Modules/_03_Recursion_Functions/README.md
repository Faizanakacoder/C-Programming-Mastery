# 📘 Recursion in C Programming

## 🚀 Introduction

Recursion is a programming technique in which a function calls itself to solve a problem.

Instead of using loops, recursion breaks a problem into smaller sub-problems until a base condition is reached.

---

## 🎯 Why Recursion is Important?

Recursion is widely used in:

- Tree and Graph algorithms 🌳
- Searching & Sorting algorithms 🔍
- Mathematical problems ➗
- Backtracking problems 🧠
- Competitive programming 💻
- Interview questions 💼

---

## 🌍 Real Life Example

Imagine standing in front of mirrors placed opposite each other 🔄

You see repeated reflections going deeper and deeper.

This is similar to recursion → same process repeating with smaller depth.
