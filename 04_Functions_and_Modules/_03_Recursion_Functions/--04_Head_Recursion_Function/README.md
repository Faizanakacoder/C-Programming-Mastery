# 📘 Head Recursion in C Programming

## 🚀 Introduction

Head recursion is a type of recursion where the **recursive call is made before any processing or execution of statements** inside the function.

In simple words:

👉 First recursive call happens  
👉 Then work is done while returning  

---

## 🎯 Why Head Recursion is Important?

Head recursion is used in:

- Tree and graph traversal 🌳  
- Backtracking problems 🔁  
- Expression evaluation  
- Stack-based processing  
- Advanced algorithm design  

---

## 🌍 Real Life Example

Think of reading a book 📖:

- You go to the last page first (recursive call phase)  
- Then you start reading backwards while returning
