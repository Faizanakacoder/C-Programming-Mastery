# 📘 Syntax of String Functions

---

# 📌 strlen()

```c
strlen(string_name);
```

---

# 📌 strcpy()

```c
strcpy(destination,source);
```

---

# 📌 strcat()

```c
strcat(string1,string2);
```

---

# 📌 strcmp()

```c
strcmp(string1,string2);
```

---

# 📌 Important Rules

✅ Include string.h header file  
✅ Use character arrays for strings  
✅ Strings end with null character '\0'  

---

# 📌 Common Mistakes

❌ Missing string.h  
❌ Using wrong string syntax  
❌ Insufficient array size
