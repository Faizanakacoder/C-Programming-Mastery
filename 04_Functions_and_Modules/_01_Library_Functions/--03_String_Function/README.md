# 📌 Introduction

String functions are predefined functions used to perform operations on strings.

These functions are available inside the string.h header file.

String functions simplify string handling and reduce coding effort.

---

# 📚 Header File Used

```c
#include<string.h>
```

---

# 📖 Functions Covered

- strlen()
- strcpy()
- strcat()
- strcmp()
