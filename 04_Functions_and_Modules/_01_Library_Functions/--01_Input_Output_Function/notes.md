# 📌 Important Header Files

| Header File | Usage |
|-------------|------|
| stdio.h | Input Output |
| math.h | Mathematics |
| string.h | String Functions |
| stdlib.h | Utility Functions |

# 📌 Most Important Functions

| Function | Work |
|----------|------|
| printf() | Output |
| scanf() | Input |
| sqrt() | Square Root |
| pow() | Power |
| strlen() | String Length |
| strcpy() | Copy String |
| strcat() | Join Strings |
| strcmp() | Compare Strings |
| rand() | Random Number |

# 📌 Viva Questions

## What is a library function?

A predefined function provided by C libraries.

---

## Why are header files important?

They contain declarations of predefined functions.

## Which header file is used for sqrt()?


## Which header file is used for strlen()?
