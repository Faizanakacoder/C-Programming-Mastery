# 📘 Syntax of Library Functions

---

# 📌 Basic Syntax

```c
#include<header_file>

int main()
{
    function_name(arguments);

    return 0;
}
```

---

# 📌 Example — printf()

```c
printf("Hello World");
```

---

# 📌 Example — scanf()

```c
scanf("%d", &n);
```

---

# 📌 Example — sqrt()

```c
sqrt(n);
```

---

# 📌 Example — strlen()

```c
strlen(str);
```

---

# 📌 Example — pow()

```c
pow(a,b);
```

---

# 📌 Important Rules

✅ Include correct header file  
✅ Use proper syntax  
✅ Use semicolon after statement  
✅ Pass correct arguments  
✅ Use correct format specifier  

---

# 📌 Common Mistakes

❌ Missing semicolon  
❌ Wrong format specifier  
❌ Missing header file  
❌ Incorrect arguments  

---

# 📌 Common Header Files

| Header File | Function Examples |
|-------------|------------------|
| stdio.h | printf(), scanf() |
| math.h | sqrt(), pow() |
| string.h | strlen(), strcpy() |
| stdlib.h | rand(), abs() |
