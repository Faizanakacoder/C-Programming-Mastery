# 📌 Introduction

Mathematical functions are predefined functions used to perform mathematical calculations easily.

These functions are available inside the math.h header file.

Instead of writing long mathematical logic manually, programmers use mathematical library functions.

---

# 📚 Header File Used

```c
#include<math.h>
```

---

# 📖 Functions Covered

- sqrt()
- pow()
- abs()
- ceil()
- floor()
- rand()
