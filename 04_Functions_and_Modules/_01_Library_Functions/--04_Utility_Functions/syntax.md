# 📘 Syntax of Utility Functions

---

# 📌 rand()

```c
rand();
```

---

# 📌 srand()

```c
srand(seed_value);
```

---

# 📌 malloc()

```c
malloc(size);
```

---

# 📌 calloc()

```c
calloc(number,size);
```

---

# 📌 free()

```c
free(pointer);
```

---

# 📌 atoi()

```c
atoi(string);
```

---

# 📌 atof()

```c
atof(string);
```

---

# 📌 Important Rules

✅ Include stdlib.h  
✅ Use pointers correctly  
✅ Release memory using free()  
✅ Use valid conversions

❌ Forgetting free()
