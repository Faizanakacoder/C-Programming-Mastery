# 📝 Quick Notes — Utility Functions

---

# 📌 Important Functions

| Function | Work |
|----------|------|
| rand() | Random Number |
| srand() | Seed Random Number |
| malloc() | Allocate Memory |
| calloc() | Multiple Memory Allocation |
| free() | Release Memory |
| atoi() | String to Integer |
| atof() | String to Float |

---

# 📌 Header File

```c
#include<stdlib.h>
```

# 📌 One Line Definitions

### rand()
Generates random number.

### malloc()
Allocates memory dynamically.

### free()

### atoi()
