# 📘 Argument with No Return Value Functions (void with parameters)

## 🚀 Introduction

In C programming, functions can accept arguments (input values) and perform operations without returning any value to the calling function.

These are called:

👉 **Argument with No Return Value Functions**

They are defined using the `void` keyword and parameters inside the function.

---

## 🎯 Why This Topic is Important?

This concept is widely used in:

- Display-based operations
- Menu-driven programs
- Printing results directly
- Modular programming
- Real-world applications where output is not needed back

---

## 🌍 Real Life Example

Think of a printer:

- You give input (document)
- It prints output
- But it does NOT return anything back

Same concept applies here 📄➡️🖨️
