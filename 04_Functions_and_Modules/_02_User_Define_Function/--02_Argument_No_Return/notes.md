# 📘 Quick Notes: Argument with No Return Value

---

## 📌 One Line Definition

Function that takes arguments but does not return value.

---

## 📌 Key Points

- Uses `void` keyword
- Accepts parameters
- No return statement
- Output inside function

---

## 📌 Important Rules

- Must pass arguments while calling
- Cannot store return value
- Used for printing operations

---

## 📌 Viva Points

- Why void is used?
- Difference between return and non-return functions
- Can we return value in void function? (No)

---

## 📌 Memory Trick

👉 “Input Yes, Output No Return”

---

## 📌 Common Uses

- Display results
- Printing tables
- Menu operations
- Simple calculations

---

## 📌 Header Files

```c
#include <stdio.h>
```

---

## 📌 Exam Tip

Always write output logic inside function.
