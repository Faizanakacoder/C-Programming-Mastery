# 📘 Syntax of No Argument No Return Functions

---

# 📌 Basic Syntax

```c
void function_name()
{
    // code
}
```

---

# 📌 Example

```c
#include<stdio.h>

void display();

void main()
{
    display();
}

void display()
{
    printf("Hello");
}
```

---

# 📌 Important Rules

✅ Use void return type  
✅ No parameters inside brackets  
✅ Function should be called properly  

---

# 📌 Common Mistakes

❌ Missing function call  
❌ Wrong syntax  
❌ Missing semicolon after declaration
