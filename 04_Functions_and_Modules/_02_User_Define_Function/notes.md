# 📝 Quick Notes — User Defined Functions

---

# 📌 Important Terms

| Term | Meaning |
|------|---------|
| Declaration | Function announcement |
| Definition | Function body |
| Call | Function execution |
| Return | Sends value back |

---

# 📚 Four Types of Functions

| Type | Arguments | Return |
|------|-----------|---------|
| No Argument No Return | ❌ | ❌ |
| Argument No Return | ✅ | ❌ |
| No Argument Return | ❌ | ✅ |
| Argument Return | ✅ | ✅ |

---

# 📌 Important Exam Points

⭐ Functions reduce repetition  
⭐ Functions improve readability  
⭐ Functions support modular programming
