# 📘 No Argument but Return Value Functions in C

## 🚀 Introduction

In C programming, functions are designed to make programs modular, reusable, and easy to manage.

One important type of function is:

👉 **No Argument but Return Value Function**

In this type:
- Function does NOT take any input (no arguments)
- But it RETURNS a value back to the calling function

This makes it useful for fixed calculations and internal processing.

## 🎯 Why This Topic is Important?

This concept is widely used in:

- Mathematical calculations
- Predefined logic functions
- Competitive programming basics
- Placement coding rounds

## 🌍 Real-Life Example

Think of a vending machine 🍫

- It returns a product/output automatically
