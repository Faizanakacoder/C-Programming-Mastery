# 📘 Arrays in C - Theory

---

## 📌 What is an Array?

An array is a collection of elements of the same data type stored in contiguous memory locations.

---

## 🧠 How Arrays Work

- Stored in continuous memory blocks
- Accessed using index starting from 0
- Fast and efficient memory usage

---

## 📊 Types of Arrays

### 1️⃣ One Dimensional Array (1D)
- Linear structure
- Single row data

### 2️⃣ Two Dimensional Array (2D)
- Matrix form
- Rows and columns

### 3️⃣ Multi-Dimensional Array
- 3D or more
- Complex data storage

---

## ⚙️ Memory Representation

Example:

int arr[5] = {10, 20, 30, 40, 50};

Memory:
Index → Value
0 → 10
1 → 20
2 → 30
3 → 40
4 → 50

---

## 🌍 Real Life Example

- Train coaches connected in sequence
- Each coach = array element
- Position = index

---

## 📌 Advantages

✔ Fast access using index  
✔ Easy traversal  
✔ Efficient memory usage  
✔ Used in algorithms  

---

## ❌ Disadvantages

❌ Fixed size  
❌ Insertion/deletion is costly  
❌ Wasted memory if unused
