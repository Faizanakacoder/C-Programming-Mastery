# 📘 Array with Functions - Theory

---

## 📌 What is Array with Functions?

It is a concept where arrays are passed as arguments to functions for processing.

---

## 🧠 How It Works

- Array is passed by reference
- Function receives base address
- Operations are performed directly on memory

---

## ⚙️ Passing Array to Function

int arr[5];

void function(int arr[])
{
    // logic
}

---

## 📊 Types of Passing Arrays

### 1️⃣ Passing Entire Array
- Used for processing full data

### 2️⃣ Passing Elements
- Used for specific operations
