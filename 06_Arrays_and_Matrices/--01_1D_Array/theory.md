# 📘 1D Array - Theory

---

## 📌 What is a 1D Array?

A 1D array is a collection of elements stored in contiguous memory locations and accessed using a single index.

---

## 🧠 Memory Representation

Example:

int arr[5] = {10, 20, 30, 40, 50};

Memory layout:
Index → Value
0 → 10
1 → 20
2 → 30
3 → 40
4 → 50

---

## ⚙️ How 1D Arrays Work

- Stored in continuous memory blocks
- Accessed using index
- Fast access time O(1)

---

## 📊 Operations on 1D Array

- Traversal
- Insertion
- Deletion
- Searching
- Sorting

---

## 🌍 Real Life Example

- Bank account transactions list
- Student marks list
- Daily temperature record

---

## 📌 Matrix Concept Using 1D Array

Even 2D matrix can be stored in 1D:

Example:
2x3 matrix → 1D array of size 6

arr[6] = {1,2,3,4,5,6}

Mapping:
(0,0) → arr[0]
(0,1) → arr[1]
(1,0) → arr[2] ... etc

---

## 📌 Advantages

✔ Simple structure  
✔ Fast access  
✔ Easy traversal  

---

## ❌ Disadvantages

❌ Fixed size  
❌ Inefficient for dynamic data
