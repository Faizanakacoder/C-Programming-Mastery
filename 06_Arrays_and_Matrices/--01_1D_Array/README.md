# 📊 1D Array in C Programming

---

## 📌 Introduction

A 1D array is a linear data structure used to store multiple elements of the same data type in a single row-like structure.

It is the simplest form of array and the foundation for all advanced array concepts like 2D arrays, matrices, and data structures.

---

## 🎯 Importance of 1D Arrays

- Used in almost every C program
- Foundation of data structures
- Used in sorting & searching algorithms
- Helps in understanding memory layout
- Required in exams + interviews

---

## 🌍 Real Life Example

Think of a **row of lockers**:

- Each locker has a number (index)
- Each locker stores a value
- Access using position
