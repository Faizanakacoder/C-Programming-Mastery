# 📘 1D Array - Syntax

---

## 📌 Declaration

int arr[5];

---

## 📌 Initialization

int arr[5] = {1, 2, 3, 4, 5};

---

## 📌 Input

for(i = 0; i < n; i++)
{
    scanf("%d", &arr[i]);
}

---

## 📌 Output

for(i = 0; i < n; i++)
{
    printf("%d ", arr[i]);
}

---

## 📌 Traversing Array

for(i = 0; i < size; i++)
{
    // logic
}

---

## 📌 Key Rules

- Index starts from 0
- Must define size
- Same data type only
- Stored in continuous memory

---

## 🚀 Summary

1D arrays are accessed using index-based looping logic.
