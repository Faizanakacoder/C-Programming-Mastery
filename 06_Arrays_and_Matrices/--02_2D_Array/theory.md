# 📘 2D Array (Matrix) - Theory

---

## 📌 What is a 2D Array?

A 2D array is an array of arrays used to store data in row-column format.

---

## 🧠 Structure

int arr[2][3];

- 2 → rows
- 3 → columns

---

## ⚙️ Memory Representation

Stored in **row-major order**:

arr[0][0], arr[0][1], arr[0][2]
arr[1][0], arr[1][1], arr[1][2]

---

## 📊 Types of Operations

- Matrix input/output
- Addition
- Subtraction
- Multiplication
- Transpose
- Diagonal operations

---

## 🌍 Real Life Example

- Spreadsheet (Excel)
- Chessboard
- Game grids
- Pixel images

---

## 📌 Advantages

✔ Structured data storage  
✔ Used in math & engineering  
✔ Easy row-column access  

---

## ❌ Disadvantages

❌ Fixed size  
❌ Complex loops required
