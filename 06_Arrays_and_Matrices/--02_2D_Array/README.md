# 📊 2D Array (Matrix) in C Programming

---

## 📌 Introduction

A 2D array in C is used to store data in **row and column format**, also known as a matrix.

It is widely used in:
- Mathematics (matrices)
- Graphics programming
- Data representation
- Machine learning basics
- Engineering calculations

---

## 🎯 Why 2D Arrays are Important?

- Used in matrix operations
- Foundation of image processing
- Used in graph problems
- Important for interviews
- Core concept in data structures

---

## 🌍 Real Life Example

Think of a **school attendance register**:

- Rows → Students
- Columns → Days
- Each cell stores attendance
