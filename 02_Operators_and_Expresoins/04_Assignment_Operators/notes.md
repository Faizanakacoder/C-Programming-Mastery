# Assignment Operators in C - Quick Notes

## Definition

Assignment operators are used to assign and update variable values.

---

## Types of Assignment Operators

| Operator | Meaning |
|-----------|----------|
| = | Assign |
| += | Add and Assign |
| -= | Subtract and Assign |
| *= | Multiply and Assign |
| /= | Divide and Assign |
| %= | Modulus and Assign |

---

## Example

```c
a += 10;
```

Equivalent to:

```c
a = a + 10;
```

---

## Important Point

Assignment operators reduce code size.

---

## Short Summary

Assignment operators are used to assign and update values in variables.
