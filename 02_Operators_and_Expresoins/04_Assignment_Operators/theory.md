# Theory - Assignment Operators in C

## Introduction

Assignment operators are used to assign values to variables.

They are also used to update values.

Example:

```c
a += 5;
```

This is equal to:

```c
a = a + 5;
```

---

## Types of Assignment Operators

### 1. Assignment (=)

Assigns value to variable.

Example:

```c
a = 10;
```

---

### 2. Add and Assign (+=)

Adds value and stores result.

Example:

```c
a += 5;
```

---

### 3. Subtract and Assign (-=)

Subtracts value and stores result.

Example:

```c
a -= 5;
```

---

### 4. Multiply and Assign (*=)

Multiplies and stores result.

Example:

```c
a *= 5;
```

---

### 5. Divide and Assign (/=)

Divides and stores result.

Example:

```c
a /= 5;
```

---

### 6. Modulus and Assign (%=)

Finds remainder and stores result.

Example:

```c
a %= 5;
```

---

## Importance of Assignment Operators

- Reduces code length
- Makes code cleaner
- Easy value updating

---

## Conclusion

Assignment operators simplify calculations and variable updates in C.
