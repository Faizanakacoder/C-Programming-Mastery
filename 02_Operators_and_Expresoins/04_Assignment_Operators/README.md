# Assignment Operators in C

## Overview

Assignment operators are used to assign values to variables.

They help perform operations and store results in variables.

Example:

```c
a += 10;
```

This means:

```c
a = a + 10;
```

---

## Types of Assignment Operators

| Operator | Meaning |
|-----------|----------|
| = | Assign |
| += | Add and Assign |
| -= | Subtract and Assign |
| *= | Multiply and Assign |
| /= | Divide and Assign |
| %= | Modulus and Assign |
