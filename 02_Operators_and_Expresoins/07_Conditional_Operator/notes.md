# Conditional Operator in C - Quick Notes

## Definition

Conditional operator is used for decision making.

Also called:

```txt
Ternary Operator
```

---

## Syntax

```c
condition ? true_statement : false_statement;
```

---

## Example

```c
age >= 18 ? printf("Eligible") : printf("Not Eligible");
```

---

## Important Point

- `?` → True part
- `:` → False part

Acts like:

```txt
if-else shortcut
```

---

## Short Summary

Conditional operator is a shortcut of `if-else`.
