# Theory - Bitwise Operators in C

## Introduction

Bitwise operators work on binary values (bits).

Computers store data in binary format (0 and 1).

Bitwise operators perform operations directly on bits.

---

## Types of Bitwise Operators

### 1. Bitwise AND (&)

Returns 1 only if both bits are 1.

Example:

```c
a & b
```

---

### 2. Bitwise OR (|)

Returns 1 if at least one bit is 1.

Example:

```c
a | b
```

---

### 3. Bitwise XOR (^)

Returns 1 if bits are different.

Example:

```c
a ^ b
```

---

### 4. Bitwise NOT (~)

Reverses bits.

Example:

```c
~a
```

---

### 5. Left Shift (<<)

Shifts bits to the left.

Example:

```c
a << 1
```

Equivalent to multiplying by 2.

---

### 6. Right Shift (>>)

Shifts bits to the right.

Example:

```c
a >> 1
```

Equivalent to dividing by 2.

---

## Importance of Bitwise Operators

- Faster operations
- Binary calculations
- Used in embedded systems
- Used in low-level programming

---

## Conclusion

Bitwise operators manipulate binary values directly.
