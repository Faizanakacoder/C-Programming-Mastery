# Bitwise Operators in C

## Overview

Bitwise operators perform operations on binary numbers (bits).

They work at the bit level.

Example:

```c
a & b
```

Here, `&` is a bitwise operator.

---

## Types of Bitwise Operators

| Operator | Meaning |
|-----------|----------|
| & | Bitwise AND |
| \| | Bitwise OR |
| ^ | Bitwise XOR |
| ~ | Bitwise NOT |
| << | Left Shift |
| >> | Right Shift |
