# Increment and Decrement Operators in C

## Overview

Increment and decrement operators are used to increase or decrease the value of a variable by 1.

Example:

```c
a++;
```

This increases the value of `a` by 1.

---

## Types of Operators

| Operator | Meaning |
|-----------|----------|
| ++ | Increment |
| -- | Decrement |

---

## Types of Increment and Decrement

1. Pre Increment (`++a`)
2. Post Increment (`a++`)
3. Pre Decrement (`--a`)
4. Post Decrement (`a--`)
