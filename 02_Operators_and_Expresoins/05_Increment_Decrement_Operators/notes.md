# Increment and Decrement Operators in C - Quick Notes

## Definition

Increment and decrement operators are used to increase or decrease values by 1.

---

## Operators

| Operator | Meaning |
|-----------|----------|
| ++ | Increment |
| -- | Decrement |

---

## Types

### Pre Increment

```c
++a
```

### Post Increment

```c
a++
```

### Pre Decrement

```c
--a
```

### Post Decrement

```c
a--
```

---

## Important Point

- Pre → Change first
- Post → Use first

---

## Short Summary

Increment and decrement operators update values quickly.
