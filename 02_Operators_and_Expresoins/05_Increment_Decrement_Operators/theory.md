# Theory - Increment and Decrement Operators in C

## Introduction

Increment and decrement operators are used to change the value of a variable by 1.

---

## Increment Operator (++)

Used to increase the value by 1.

Example:

```c
a++;
```

Equivalent to:

```c
a = a + 1;
```

---

## Decrement Operator (--)

Used to decrease the value by 1.

Example:

```c
a--;
```

Equivalent to:

```c
a = a - 1;
```

---

## Pre Increment (++a)

Value increases first, then used.

Example:

```c
++a;
```

---

## Post Increment (a++)

Value used first, then increases.

Example:

```c
a++;
```

---

## Pre Decrement (--a)

Value decreases first, then used.

Example:

```c
--a;
```

---

## Post Decrement (a--)

Value used first, then decreases.

Example:

```c
a--;
```

---

## Importance

- Shorter code
- Faster value updates
- Commonly used in loops

---

## Conclusion

Increment and decrement operators help modify values quickly in C programming.
