# Arithmetic Operators in C - Quick Notes

## Definition

Arithmetic operators are used to perform mathematical calculations.

---

## Types of Arithmetic Operators

| Operator | Meaning |
|-----------|----------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Modulus |

---

## Examples

### Addition

```c
a + b
```

### Subtraction

```c
a - b
```

### Multiplication

```c
a * b
```

### Division

```c
a / b
```

### Modulus

```c
a % b
```

---

## Important Point

`%` works only with integers.

---

## Short Summary

Arithmetic operators are used to perform mathematical operations in C.
