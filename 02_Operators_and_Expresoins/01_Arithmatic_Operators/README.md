# Arithmetic Operators in C

## Overview

Arithmetic operators are used to perform mathematical calculations in C programming.

These operators work with numbers and variables.

Example:

```c
sum = a + b;
```

Here, `+` is an arithmetic operator.

---

## Types of Arithmetic Operators

| Operator | Meaning |
|-----------|----------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Modulus |
