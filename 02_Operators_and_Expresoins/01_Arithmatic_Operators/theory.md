# Theory - Arithmetic Operators in C

## Introduction

Arithmetic operators are used to perform mathematical operations on numbers.

These operators help in calculations like addition, subtraction, multiplication, division, and remainder.

Example:

```c
a + b
```

Here, `+` is an arithmetic operator.

---

## Types of Arithmetic Operators

### 1. Addition Operator (+)

Used to add two values.

Example:

```c
sum = a + b;
```

---

### 2. Subtraction Operator (-)

Used to subtract values.

Example:

```c
difference = a - b;
```

---

### 3. Multiplication Operator (*)

Used to multiply values.

Example:

```c
product = a * b;
```

---

### 4. Division Operator (/)

Used to divide values.

Example:

```c
division = a / b;
```

---

### 5. Modulus Operator (%)

Used to find remainder.

Example:

```c
remainder = a % b;
```

---

## Importance of Arithmetic Operators

- Used in calculations
- Helps solve mathematical problems
- Makes programs interactive

---

## Real Life Example

Calculator apps use arithmetic operators:

- `+` → Addition
- `-` → Subtraction
- `*` → Multiplication
- `/` → Division

---

## Conclusion

Arithmetic operators are essential for performing mathematical calculations in C programming.
