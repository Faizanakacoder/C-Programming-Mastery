# Theory - Relational Operators in C

## Introduction

Relational operators are used to compare two values.

These operators help in decision making and conditions.

The output of relational operators is:

- `1` → True
- `0` → False

Example:

```c
10 > 5
```

Output:

```txt
1
```

---

## Types of Relational Operators

### 1. Equal To (==)

Checks if two values are equal.

Example:

```c
a == b
```

---

### 2. Not Equal To (!=)

Checks if values are not equal.

Example:

```c
a != b
```

---

### 3. Greater Than (>)

Checks if left value is greater.

Example:

```c
a > b
```

---

### 4. Less Than (<)

Checks if left value is smaller.

Example:

```c
a < b
```

---

### 5. Greater Than or Equal To (>=)

Checks greater or equal condition.

Example:

```c
a >= b
```

---

### 6. Less Than or Equal To (<=)

Checks smaller or equal condition.

Example:

```c
a <= b
```

---

## Importance of Relational Operators

- Used in conditions
- Helps decision making
- Used with `if`, `while`, and loops

---

## Real Life Example

Age verification:

```c
age >= 18
```

If true → person can vote.

---

## Conclusion

Relational operators compare values and help make decisions in programs.
