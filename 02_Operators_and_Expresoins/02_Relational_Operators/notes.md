# Relational Operators in C - Quick Notes

## Definition

Relational operators are used to compare two values.

Output:

- `1` → True
- `0` → False

---

## Types of Relational Operators

| Operator | Meaning |
|-----------|----------|
| == | Equal To |
| != | Not Equal To |
| > | Greater Than |
| < | Less Than |
| >= | Greater Than or Equal To |
| <= | Less Than or Equal To |

---

## Example

```c
10 > 5
```

Output:

```txt
1
```

---

## Important Point

Relational operators always return:

```txt
True → 1
False → 0
```

---

## Short Summary

Relational operators compare values in C programming.
