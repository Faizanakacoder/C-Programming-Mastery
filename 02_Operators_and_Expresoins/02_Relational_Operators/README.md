# Relational Operators in C

## Overview

Relational operators are used to compare two values or variables.

They return:

- `1` → True
- `0` → False

Example:

```c
a > b
```

Here, `>` is a relational operator.

---

## Types of Relational Operators

| Operator | Meaning |
|-----------|----------|
| == | Equal to |
| != | Not Equal to |
| > | Greater Than |
| < | Less Than |
| >= | Greater Than or Equal To |
| <= | Less Than or Equal To |
