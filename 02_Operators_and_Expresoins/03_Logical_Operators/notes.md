# Logical Operators in C - Quick Notes

## Definition

Logical operators are used to combine conditions.

Output:

- `1` → True
- `0` → False

---

## Types of Logical Operators

| Operator | Meaning |
|-----------|----------|
| && | Logical AND |
| || | Logical OR |
| ! | Logical NOT |

---

## Example

```c
age >= 18 && citizen == 1
```

---

## Important Point

Logical operators always return:

```txt
True → 1
False → 0
```

---

## Short Summary

Logical operators are used to combine multiple conditions in C.
