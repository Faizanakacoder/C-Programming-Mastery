# Logical Operators in C

## Overview

Logical operators are used to combine conditions in C programming.

They return:

- `1` → True
- `0` → False

Logical operators are mainly used with `if` statements and decision-making.

Example:

```c
age >= 18 && citizen == 1
```

---

## Types of Logical Operators

| Operator | Meaning |
|-----------|----------|
| && | Logical AND |
| || | Logical OR |
| ! | Logical NOT |
