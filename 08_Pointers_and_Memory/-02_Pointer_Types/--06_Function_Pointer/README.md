# 🎯 Function Pointer in C Programming

---

# 📌 Introduction

A **Function Pointer** is a pointer that stores the address of a function.

Just like:

* int pointer → stores int address
* char pointer → stores char address
* function pointer → stores function address

---

# 📌 Syntax Idea

return_type (*pointer_name)(parameters);

---

# 🎯 Why Function Pointer is Important?

Function pointers are used in:

* callback functions
* dynamic function calling
* menu-driven programs
* event handling
* advanced system programming

---

# 🌍 Real Life Usage
