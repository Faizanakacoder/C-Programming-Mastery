# 🧠 Pointer Types in C Programming

---

# 📌 Introduction

Pointers in C can point to different types of data.

Depending on the datatype they store addresses for,
pointers are categorized into multiple types.

Examples:

* Integer Pointer
* Character Pointer
* Float Pointer
* Double Pointer
* Void Pointer
* Function Pointer

Each pointer type:
works with a specific datatype.

---

# 🎯 Why Pointer Types are Important?

Pointer types help:
✅ access correct memory data
✅ maintain datatype safety
✅ improve program readability
✅ avoid invalid memory access

Understanding pointer types is important for:

* arrays
* functions
* structures
* dynamic memory allocation
* advanced C programming
