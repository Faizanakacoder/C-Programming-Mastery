# 🔁 Double Pointer in C Programming

---

# 📌 Introduction

A Double Pointer is a pointer that:
stores the address of another pointer.

It is also called:

✅ Pointer to Pointer
✅ Nested Pointer

Example:

int num = 10;

int *ptr = #

int **dptr = &ptr;

Here:

* `num` stores integer value
* `ptr` stores address of num
* `dptr` stores address of ptr

Double pointers are used in:
✅ dynamic memory allocation
✅ arrays of pointers
✅ function arguments
✅ advanced memory handling

---

# 🎯 Why Double Pointer is Important?

Double pointers are important because:

✅ They manage pointers dynamically
✅ Used in advanced memory allocation
✅ Required in linked data structures
✅ Useful in multidimensional arrays
✅ Important for professional C programming
