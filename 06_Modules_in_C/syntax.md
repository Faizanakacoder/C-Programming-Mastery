# 📘 Modules in C Programming - Complete Syntax Reference

---

## 🚀 Overview

Modules in C are implemented using **multiple files, functions, and headers**. This syntax file acts as a **quick revision + interview + viva + placement cheat sheet** for modular programming.

Modular programming mainly uses:

- `.c` files → Implementation
- `.h` files → Declarations
- `main.c` → Execution control
- Compiler linking → Combines modules

---

# 📌 1. Single File Module Syntax

A simple program where everything exists in one file.

```c
#include<stdio.h>

int main()
{
    // program logic
    printf("Hello Modular World");

    return 0;
}
